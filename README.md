# Cybersecurity Internship – OWASP Juice Shop

**Student:** Muhammad Shahid    

## Project Overview
This project demonstrates security assessment and implementation on OWASP Juice Shop, a vulnerable web application. Tasks include vulnerability scanning, fixes, and logging.

## Tools Used
Node.js, npm, VS Code, PowerShell, XAMPP, Chrome, OWASP ZAP, Nmap, Validator, Bcrypt, JWT, Helmet.js, Winston

## Project Files
- `SECURITY_CHECKLIST.md` – Security measures and best practices  
- `Final_Report.pdf` – Full internship report  
- Juice Shop code (project source files)

## Steps Performed
1. Setup Juice Shop (npm install, npm start)  
2. Week 1: Security assessment using ZAP and manual tests  
3. Week 2: Input validation, password hashing, JWT, Helmet.js  
4. Week 3: Nmap testing, Winston logging, checklist creation
